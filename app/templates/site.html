Language Model Lesions

This is a visualization of the alpaca large language model. In this image it has no lesions applied, the dark portion at the beginning is the models vocabulary, that is all the words and sub-word that the model can use. The colourful portion is the weights, this is the black box where relationships in data are learned, stored, and used to make predictions. This is zoomed in the better show the weights in all their deceptive randomness.

In the following slides you will see just the weights, with black bars across. These bars are lesions, that is data that has been removed from the model file before it was ran and asked the prompting questions.

The code used to create these slides and the images is hosted here: https://github.com/signed-long/lml
